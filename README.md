# EcomPulse Chrome 插件

EcomPulse 是一款基于Chrome浏览器的智能购物助手插件， 旨在帮助消费着实现**跨平台价格透明化**。 通过自动化抓取主流电商历史价格数据并实现全网比价，为用户提供给科学的购物决策支持，避免`先涨后降`的消费陷阱，节省时间和金钱
